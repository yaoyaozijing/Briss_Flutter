# ProCropper PDF

Flutter で構築されたクロスプラットフォーム対応の PDF トリミングツールです。モバイルとデスクトップの両方で、よりスマートかつ効率的な PDF 余白トリミング体験を提供することを目指しています。追加の実行環境に依存せず、単体アプリとして動作します。

本プロジェクトは、Briss の重ね合わせプレビューとグループ単位のトリミングという考え方を参考にし、その上で最適化と改良を行っています。
システムはページ内容（文字やインクの分布）に基づいてページ構造を自動解析し、ページを複数のトリミンググループに分類して対応するトリミング領域を生成できるため、ユーザーの手動操作を大幅に減らせます。

# 主な機能

- [x] PDF の重ね合わせプレビュー
- [x] 複数のインテリジェントなページグループ化戦略
- [x] トリミング枠の自動生成
- [x] トリミング枠の手動調整
- [x] 複数ページ間でのトリミング枠のコピー＆ペースト
- [x] 1 つのグループに複数のトリミング枠を追加し、ページ分割のような処理に対応
- [x] ページ端領域の手動フィルタリング
- [x] 縦横比の固定により、さまざまな画面サイズで全画面体験を提供
- [x] 外側に余白を広げるモード
- [x] モバイル端末での共有経由 PDF インポート／エクスポート
- [x] デスクトップ端末でのパラメータ指定およびドラッグ＆ドロップによるインポート
- [x] ホーム画面でのドラッグ＆ドロップによる PDF インポート
- [x] 大画面・小画面の両方に対応したレスポンシブなエディター画面
- [x] 多言語対応（中国語、英語、日本語）
- [x] E-ink ディスプレイ最適化
- [x] 暗号化された PDF の処理
- [x] トリミング結果のプレビュー
- [x] マルチウィンドウ対応（実験的）
- [x] 全自動バッチトリミング
- [ ] パフォーマンス最適化と大容量ファイル対応の強化
